# Reax — EPL 팬 반응 피드

> **"트윗 하나가 논쟁이 된다"**
> 해외 이적 루머·오피셜을 가장 빠르게 확인하고, 팬 여론까지 한 번에 읽는 서비스

---

## 1. 한 줄 소개

**Reax**는 EPL 이슈를 중심으로, AI가 실시간 트윗에서 논쟁을 자동 생성하고 팬 여론을 요약해주는 모바일 피드 서비스다.

---

## 2. 문제 정의

### EPL 팬이 현재 겪는 불편함

| 행동 | 문제 |
|---|---|
| 이적 루머 확인 | 트위터, 네이버, 더쿠, DC인사이드 등 4~5개 채널을 돌아다녀야 한다 |
| 팬 여론 파악 | 수백 개 댓글을 직접 읽어야 "지금 여론이 어느 쪽인지" 감이 잡힌다 |
| 논쟁 참여 | 어디서 말해야 할지 모른다. 커뮤니티는 진입 장벽이 높다 |
| 정보 신뢰도 판단 | 루머인지 오피셜인지, 소스가 믿을 만한지 매번 확인해야 한다 |

**핵심 인사이트**: EPL 팬들은 "이슈 확인"과 "여론 확인"을 동시에 원하지만, 지금은 이 두 가지가 완전히 분리되어 있다.

---

## 3. 솔루션

### Reax가 하는 것

```
트윗 (Fabrizio Romano, David Ornstein 등)
        ↓
   AI 자동 분석
        ↓
 ┌──────────────────────────┐
 │  HOT DEBATE 카드 생성     │
 │  • 논쟁 질문 자동 생성    │
 │  • 찬반 여론 요약         │
 │  • 실시간 투표            │
 │  • 팬 댓글 큐레이션       │
 └──────────────────────────┘
        ↓
  TikTok 스타일 세로 피드로 소비
```

이슈 확인 → 여론 파악 → 논쟁 참여까지, 한 앱에서 끝난다.

---

## 4. 핵심 차별점

### 4-1. AI-First 여론 요약

**기존 서비스의 방식**: 댓글/투표를 먼저 유도하고, 참여자만 요약을 볼 수 있다.

**Reax의 방식**: AI가 현재 여론을 먼저 요약해서 보여준다. 관심이 생긴 사람이 투표·논쟁에 참여한다.

> "나한테 관심 없는 논쟁에 먼저 투표해야 내용을 볼 수 있다"는 장벽을 제거한다.

```
기존: [투표] → [여론 잠금 해제] → [요약 확인]
Reax: [AI 여론 요약] → [관심 생기면 투표 참여]
```

---

### 4-2. 트윗 → 논쟁 자동 생성

Fabrizio Romano, David Ornstein 등 신뢰도 높은 기자의 트윗을 소스로, AI가:

- 논쟁 질문을 자동 생성
- 찬반 핵심 논점을 정리
- 현재 팬 여론의 분포를 시각화

**큐레이션이 아니다** — AI가 트윗을 보고 논쟁을 만드는 것이다. 이 흐름을 카드에 `⚡ 이 트윗에서 생성된 논쟁` 레이블로 명시한다.

---

### 4-3. 소스 신뢰도 레이어

모든 트윗에 **T1 / T2** 티어 표시:

- **T1**: Fabrizio Romano, David Ornstein, Sky Sports 등 — 업계 최상위 소스
- **T2**: 지역 기자, 2차 소스 등

팬이 정보를 볼 때 즉시 신뢰도를 판단할 수 있다.

---

### 4-4. 팀 퍼스널라이제이션

온보딩에서 좋아하는 팀을 선택하면:

- 피드에서 내 클럽 관련 포스트에 **"내 클럽"** 배지 표시
- 내 클럽 포스트가 피드 상단에 우선 배치
- 하단 내비게이션의 MY 아이콘에 팀 컬러 적용

단순한 뉴스 앱이 아니라, **내 팀을 응원하는 팬의 앱**이 된다.

---

### 4-5. 모바일 퍼스트 소비 경험

TikTok / Instagram Reels 방식의 **세로 스크롤 풀스크린 피드**:

- 카드 하나 = 이슈 하나 = 선수/경기 사진 배경 + 요약 + 트윗 + 찬반 바
- 스와이프 한 번으로 다음 이슈
- 복잡한 메뉴 없이 엄지 하나로 모두 소화

데스크탑 커뮤니티 포맷이 아닌, 이동 중 30초 안에 소비 가능한 형태.

## 5. 주요 기능

### 피드 (메인)
- 이슈 카드 세로 스크롤
- HOT DEBATE / TODAY DEBATE / 일반 정보 / 감성 포스트 4가지 타입
- 카드별 선수 실사 사진 배경 + 다크 오버레이

### HOT DEBATE
- 찬반 투표 (투표 전도 AI 요약 노출)
- 찬성/반대 핵심 논점 카드
- 베스트 댓글 / 실시간 댓글 탭 전환
- 트렌드 표시 (최근 30분 여론 변화 방향)

### HOT 탭
- 현재 논쟁 중인 DEBATE 포스트만 모아보기

### MY 탭
- 내가 선택한 클럽 홈 카드 (팀 컬러, 순위, 최근 결과)
- 내 투표 기록
- 알림 설정

---

## 6. 타겟 사용자

### 1차 타겟
**EPL을 즐겨 보는 20~35세 남성**

- 이적 시즌마다 트위터·더쿠·DC 새로고침하는 사람
- 경기 후 팬 반응이 궁금해서 커뮤니티를 뒤지는 사람
- 특정 클럽의 팬 (맨유, 아스날, 맨시티, 첼시, 리버풀, 토트넘)

### 핵심 사용 시나리오
1. **출퇴근길**: 어젯밤 이적 루머 + 팬 여론을 5분 안에 파악
2. **경기 전후**: HOT DEBATE로 오늘의 논쟁 확인, 한 마디 참여
3. **이적 시즌**: 매일 쏟아지는 루머·오피셜을 한 곳에서 필터링

---

## 7. 왜 지금인가

### EPL의 글로벌 성장
- EPL은 전 세계 180개국 방영, 아시아 팬베이스 급성장
- 한국에서도 주요 구단 팬 커뮤니티가 수십만 규모

### AI 기술의 성숙
- 트윗 → 논쟁 자동 생성 / 여론 요약이 실용 수준에 도달
- 이 기능을 MVP에 녹이는 것이 현실적으로 가능해진 시점

### 기존 서비스의 공백
- 국내에는 EPL 특화 팬 반응 플랫폼이 없다
- 해외 서비스(Goal, OneFootball)는 뉴스 중심, 여론 참여 기능 없음
- 커뮤니티(더쿠, DC)는 진입 장벽 + 모바일 UX 취약

---

## 8. 디자인 철학

### "에디토리얼 다크 피드"
- 어두운 배경 + 선수 실사 사진 배경 → 몰입감
- 빨간(DEBATE) / 노란(TODAY) 액센트 컬러로 긴장감 전달
- 과도한 UI 요소 배제, 텍스트와 이미지만으로 완결

### "먼저 보여주기"
- 투표·댓글 참여 전에도 요약을 보여줌
- 진입 장벽을 낮춰 더 많은 팬이 여론 형성에 기여하게 함

### "신호 대 잡음"
- T1/T2 소스 구분으로 루머와 오피셜 명확히 분리
- OFFICIAL / ADVANCED / TALKS / RUMOUR 상태 배지

---

## 9. 프로토타입 현황

- **프레임워크**: React 18 + Vite + TailwindCSS
- **포스트 수**: 13개 (HOT DEBATE, TODAY DEBATE, 일반, 감성 타입 믹스)
- **구현된 기능**:
  - 세로 스크롤 풀스크린 피드
  - 팬 반응 패널 (투표, AI 요약, 댓글)
  - 팀 선택 온보딩 (6페이지)
  - 하단 내비: 피드 / HOT / 검색 / MY
  - MY 탭 (클럽 홈, 내 활동, 알림 설정)
  - 선수 실사 사진 배경 카드
